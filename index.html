Introduction

  • My primary area of interest is software security. I'm particularly focusing on scalable vulnerability detection.
    • Xint: Black Box Security Testing for Web Applications
    • ChainLight WARD: Static Analysis for Solidity Smart Contracts
    • Rudra: Static Analysis for Rust Memory Safety Issues
  • I believe one's programming principles reveal a lot about how they approach engineering. For me, they are:
    1. Parse, Don't Validate: make invalid states unrepresentable
    2. You Ain't Gonna Need It (YAGNI): do not build a thing until it is necessary
    3. Programming as Theory Building: programming is about building a shared mental model
